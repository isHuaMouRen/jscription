# Console 系列命令

[主页](/docs/home.md)

Console 系列命令用于控制控制台

## console.write

在控制台打印文字

|参数|类型|描述|
|-|-|-|
| `message` |string?|输出的内容|

## console.writeline

在控制台打印一行文字（即在输出内容后方添加换行符）

|参数|类型|描述|
|-|-|-|
| `message` |string?|输出的内容|

## console.readline

获得用户输入，在用户输入完后点击回车，返回用户输入内容

|返回值类型|描述|
|-|-|
| `string` |用户输入的内容|
