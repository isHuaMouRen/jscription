# 1.0.0-indev.3

## 更新

* **命令**
  + 更新命令 `variable.get` 用于读取某个变量的值并返回

## 优化

* **命令**
  + 优化 `control.loop` 命令的性能以及修复潜在的问题
  + 优化 `control.if` 命令的性能以及修复潜在的问题
  + 优化 `control.if` 与 `variable.get` 命令执行时的错误报告
  + 使用更加强大的正则表达式优化条件判断功能
* **错误报告**: 现在运行时错误会显示命令的具体名称，例如 `console.printline` 而非 `PrintLine`

## 修复

* **变量系统**: 修复无法解析像 `$var1$ and $var2$` 这样的变量引用
