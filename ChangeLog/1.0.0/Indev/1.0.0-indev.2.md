# 1.0.0-indev.2

## 更新

* **命令**
  + 新增命令 `file.delete` 可用于删除文件
  + 新增命令 `file.read` 可用于读取文件，可搭配return返回值来给变量赋值
  + 新增流程控制命令 `control.sleep` 可用于延迟一段时间后再执行后方代码
  + 新增流程控制命令 `control.if` 可用于条件判断并选择性执行 then 与 else 处的命令
  + 新增流程控制命令 `control.loop` 可用于条件判断并重复执行 do 内的命令
  + 新增属性 `return` ，此处可填写变量名，将命令的返回值赋值给变量
* **变量系统**
  + 现在，你可以在根处添加变量，并在命令参数处以 `$变量名$` 的方式调用
  + 新增命令 `variable.set` 来修改变量

## 优化

* **命令**
  + 优化大小写转换，现在对大小写不敏感
  + 现在报错会显示其父级名称，而非一个辨识度较低的单一子级名称

## 修复

* **命令**: 在命令类型转换错时，不再只输出 `CmdRoot` ，而是真是的命令名称
