# 1.0.0-indev.5

## 更新

### 语法

这是当前的语法:

```json
{
    "command": "control.loop",
    "arguments": {
        "condition": true,
        "do": [
            {
                "command": "console.printline",
                "arguments": {
                    "message": "Hello world!"
                },
                "return": ""
            }
        ]
    }
}
```

问题是 `command` 和 `arguments` 关键字出现太频繁

于是我们升级了语法

```json
{
    "control.loop": {
        "condition": true,
        "do": [
            {
                "console.printline": {
                    "message": "Hello world!"
                },
                "return": ""
            }
        ]
    }
}
```

无需多次输入 `command` 与 `arguments` 了！

此次升级不影响旧语法，你仍然可以使用旧语法并正常运行。不过旧语法会在 **1.0.0正式版** 中被彻底移除，建议从现在开始使用新语法
