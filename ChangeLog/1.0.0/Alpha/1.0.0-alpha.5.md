# 1.0.0-alpha.5

## 更新

* **命令**
  + 添加分类 `string` 用于字符串操作
  + 添加命令 `string.replace` 用于替换字符串内某字符或字符串
  + 添加命令 `string.trim` 用于去除头尾空格，整理字符串
  + 添加命令 `string.contains` 用于判断字符串是否包含某字符串
  + 命令 `process.start` 添加参数 `waitForExit (bool?)` 用于控制是否等待进程退出
  + 命令 `process.start` 添加返回值 (int) 是进程的退出码

## 修复

* **返回值**: 修复了在参数内带有返回值的命令会执行两次的问题
