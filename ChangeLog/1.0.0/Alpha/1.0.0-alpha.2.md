# 1.0.0-alpha.2

## 更新

* **命令**
  + 更新分类 `dir`，用于文件夹操作
  + 更新命令 `file.exists`，用于检测文件是否存在
  + 更新命令 `dir.create`，用于创建文件夹
  + 更新命令 `dir.delete`，用于删除文件夹
  + 更新命令 `dir.exists`，用于检测文件夹是否存在
* **返回值系统**: 现在支持直接把命令填写到参数中！不需要再写变量、取变量了！(详见文档中的编写方法)；旧语法并不会被删除，而是作为另一种方式存在

## 优化

* **错误报告**: 在 `control.if` 或 `control.loop` 中的子命令发生错误时，其位置报告不再为if或loop父命令，而是更加精确的子命令
