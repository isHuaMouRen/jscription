# 1.0.0-alpha.1

> [!note]
> 经历Indev版本的开发，基本功能已完毕。现进入Alpha版本进行扩展开发与问题修复

## 更新

* **命令**
  + 添加命令 `console.readline` ，用于读取用户输出并返回
  + 添加命令 `console.setcolor` ，用于设置控制台前景色
  + 添加命令 `console.getcolor` ，用于获得当前控制台前景色

## 重构

* **解析**: `control.if` 与 `control.loop` 内部的命令不再手动解析，而是由主解析器进行解析。这两条命令仅起到执行逻辑

## 更改

* **命令**
  + 更改 `console.print` 的签名为 `console.write`
  + 更改 `console.printline` 的签名为 `console.writeline`
